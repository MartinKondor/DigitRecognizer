## Getting Started

### Prerequisites

* Python 3.8+
* Anaconda 4.13+ (optional)
* Python modules from the `requirements.txt`

### Deployment

Dowload and install the dependencies with the command:

```
$ python -m pip install -r requirements.txt
```

### Training

To train the model you should run:

`$ python model.py`
